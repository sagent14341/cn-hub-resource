# cn-hub-resource

本项目是一个用于归档和发布多个独立 HTML 页面的资源仓库。仓库内容不针对任何特定网站或域名，主要用于存放各类静态页面、工具页面、信息展示页面等。

## 项目简介

cn-hub-resource 旨在提供一个集中存放与发布 HTML 页面的空间。所有页面均以独立文件形式存在，不依赖特定后端服务，可直接通过浏览器访问。仓库内容适合用于展示、测试、存档或分享静态网页资源。

## 目录说明

```
cn-hub-resource/
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 页面所需的静态资源（CSS、JS、图片等）
├── README.md       # 本文件
└── LICENSE         # 项目许可证
```

- `pages/`：每个 HTML 文件代表一个独立的页面，文件名反映页面内容或用途。
- `assets/`：按需存放页面引用的样式、脚本或媒体文件，保持目录结构清晰。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在本地或托管环境中打开。
- 页面之间相互独立，不共享状态或数据。
- 归档内容会不定期更新，新增或调整页面。
- 每个页面文件头部可包含简单的注释说明，便于识别用途。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来建议新增页面或修正现有内容。
- 新增页面请放入 `pages/` 目录，并确保文件名简洁明了。
- 如需引用外部资源，建议优先使用相对路径或存放在 `assets/` 中。
- 请勿在页面中包含恶意代码、跟踪脚本或违反社区规范的内容。
